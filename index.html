關於我

介紹自己

大家好，我是易佩蓁，我是一個個性內斂、比較不擅長社交的人，其實我很喜歡認識新朋友，雖然有時會因為害羞而不太敢主動與人接觸，但我給自己立下了一個目標，希望在大學期間能勇敢面對人際關係這個課題， 我相信在人與人之間的互動中，不論是學術性的合作還是同儕之間的交流，都是寶貴的學習機會，透過這些互動，我希望能學習如何更自在地與人相處，並逐漸改變自己過去比較封閉的習慣。

此外，我也希望在大學階段培養更有主見的思考能力，在課堂上或閱讀時，我會提醒自己不要盲目接受知識，而是學習主動提問與思考，同時，我也會積極參與小組討論，練習傾聽不同的觀點，在吸收他人意見的同時，也能堅定或修正自己的想法 ，我期待透過大學生活的各種經驗，讓自己變得更有自信，也更能清楚表達自己的想法與立場。

結果說明

使用 Python 搭配 Selenium 來模擬使用者在旅遊網站「旅行圖中」上輸入關鍵字進行搜尋，並擷取搜尋結果的標題與連結程式首先透過 webdriver_manager啟動 Chrome 瀏覽器，接著模擬使用者在搜尋欄位中輸入「日本」進行搜尋，為了確保搜尋結果載入完成，程式中加入了 time.sleep() 等待時間， 接下來透過定位搜尋結果項目所使用的 HTML class 名稱，逐一擷取每筆資料的標題與網址，並存入一個字典列表中，最後程式使用 pandas 將這些資料轉存為 CSV 檔案。
